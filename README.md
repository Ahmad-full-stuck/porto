# Vaultly — صفحة هبوط لإدارة الثروة بالذكاء الاصطناعي

صفحة هبوط عربية (RTL) فاخرة لمنتج **Vaultly**: مدير ثروة شخصي يعمل بالذكاء الاصطناعي.
مبنية بجودة قالب قابل لإعادة البيع، مع نظام تصميم زجاجي حقيقي وتصميم صوت تفاعلي.

## التقنيات

- **Next.js 14** (App Router) + **TypeScript**
- **Tailwind CSS** — نظام ألوان مخصص بالكامل
- **Framer Motion** — حركة الظهور، Parallax، والطفو
- **Howler.js** — ثلاثة مؤثرات صوتية مولّدة برمجيًا (WAV Data URI، بدون أصول ثنائية)

## التشغيل المحلي

```bash
npm install
npm run dev
```

ثم افتح `http://localhost:3000`.

للبناء الإنتاجي:

```bash
npm run build && npm start
```

## تصميم الصوت

- **نقرة** خفيفة على كل CTA وعناصر التحكم.
- **تأكيد نجاح** (نغمتان صاعدتان) عند اكتمال التسجيل في قائمة الانتظار.
- **رنين زجاجي** عند انتهاء العدّ التصاعدي في بطاقة الرصيد.
- مستوى الصوت منخفض افتراضيًا (0.12–0.18)، **زر كتم** واضح في الشريط العلوي (يُحفظ في localStorage)،
  و**لا تشغيل تلقائي** قبل أول تفاعل من المستخدم.

## إمكانية الوصول

- `prefers-reduced-motion`: تعطيل Parallax والطفو والحركات، وإظهار الأرقام النهائية فورًا.
- تنقّل كامل بلوحة المفاتيح مع حالة `focus-visible` ذهبية واضحة ورابط "تخطي إلى المحتوى".
- Labels مرتبطة بكل الحقول والمؤشرات، و`aria-live` لرسائل النموذج والأرقام المتغيرة.

## API قائمة الانتظار

`POST /api/waitlist` مع `{ name?, email }`:

- يتحقق من صحة البريد ويعيد `400` مع رسالة عربية عند الخطأ.
- يمنع التسجيل المكرر ويعيد ترتيب المشترك في القائمة.
- يخزّن البيانات في `data/waitlist.json` على الخادم.
